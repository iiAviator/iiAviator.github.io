AviateCSS Documentation

Themes

The two main themes are main and light. Dark is the theme this webpage is using and light is a light version of this theme. This class (av-main or av-light) is added to the body element.

Buttons

Add the class: av-bt- and then adding one of the colors makes buttons.

Round

Adding the class av-round makes rounded corners.

Text Input

Adding the class av-txt-in creates stylized text boxes.

Alerts

Adding the class av-al- and then a certain color creates alerts.

Containers

Containers can be created by using av-container or av-cn.

Rows

Rows can be created by using av-row or av-rw.

Padding

Padding can be quickly added using av-pd1 to av-pd4

Margins

Margins can be quickly added using av-mg1 to av-mg4
